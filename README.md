# Sistema de Consultas - TypeScript

Projeto desenvolvido em TypeScript para simular um sistema simples de gerenciamento de consultas médicas.

## 📚 Objetivo

Praticar conceitos fundamentais de TypeScript, como:

- Tipagem com `type`
- Uso de `interface`
- Organização de código em módulos
- Manipulação de arrays tipados
- Funções com parâmetros tipados
- Compilação de TypeScript para JavaScript
